We build libraries with Rust cores and ergonomic bindings for Python, Java, and beyond — so you get native speed without giving up the language you already ship in.

## Libraries

| Project | Description | Stack |
| --- | --- | --- |
| **[taskito](https://github.com/ByteVeda/taskito)** | Rust-powered task queue for Python — no broker required. | Rust · Python |
| **[paperjam](https://github.com/ByteVeda/paperjam)** | One Rust engine for PDF, DOCX, XLSX, PPTX, HTML, and EPUB. | Rust · Python |
| **[agenteval](https://github.com/ByteVeda/agenteval)** | JUnit 5-native evaluation for Java AI agents. | Java |
| **[reclink](https://github.com/ByteVeda/reclink)** | High-performance fuzzy matching and record linkage. | Rust · Python |
| **[dagron](https://github.com/ByteVeda/dagron)** | Rust-backed DAG execution engine for Python. | Rust · Python |
